# 基于SpringBoot与Vue的智能仓库管理系统设计与实现

## 摘    要

随着电子商务和制造业的快速发展，企业对仓库管理的要求越来越高。传统的手工记录方式不仅效率低下，而且容易出现数据错误，难以满足现代企业对仓库管理精细化、信息化的需求。为了解决这些问题，本文设计并实现了一套基于SpringBoot与Vue的智能仓库管理系统。

本系统采用前后端分离的架构设计。后端使用SpringBoot框架搭建，结合MyBatis-Plus持久层框架实现数据库操作，使用MySQL数据库存储业务数据。前端采用Vue 3框架开发，使用Vite作为构建工具，引入Naive UI组件库提升用户界面的美观性和易用性，通过ECharts图表库实现数据可视化展示。系统实现了用户管理、供应商管理、物料管理、仓位管理、入库管理、出库管理和数据统计等核心功能模块。

在功能实现方面，系统支持物料信息的录入和维护，可以设置库存预警值，当物料库存低于预警值时自动提醒管理者及时补货。入库和出库操作会自动更新物料的库存数量，保证了数据的实时性和准确性。系统还提供了数据统计功能，通过图表直观展示出入库趋势和库存分布情况，帮助管理者更好地掌握仓库运营状态。

经过功能测试和实际运行验证，系统各项功能运行稳定，界面操作简便，能够满足中小型企业仓库管理的基本需求。本系统的设计与实现不仅提升了仓库管理的效率和准确性，也为企业的信息化建设提供了参考方案。

**关键词：仓库管理系统；SpringBoot；Vue；前后端分离；库存管理**

## ABSTRACT

With the rapid development of e-commerce and manufacturing industry, enterprises have increasingly higher requirements for warehouse management. Traditional manual recording methods are not only inefficient but also prone to data errors, making it difficult to meet the needs of modern enterprises for refined and informatized warehouse management. To solve these problems, this paper designs and implements an intelligent warehouse management system based on SpringBoot and Vue.

The system adopts a front-end and back-end separation architecture. The back-end is built using the SpringBoot framework, combined with the MyBatis-Plus persistence framework to implement database operations, and uses MySQL database to store business data. The front-end is developed using the Vue 3 framework, uses Vite as the build tool, introduces the Naive UI component library to improve the aesthetics and usability of the user interface, and implements data visualization through the ECharts chart library. The system implements core functional modules such as user management, supplier management, material management, location management, inbound management, outbound management, and data statistics.

In terms of functional implementation, the system supports the entry and maintenance of material information, and can set inventory warning values. When the material inventory is lower than the warning value, it automatically reminds managers to replenish goods in time. Inbound and outbound operations automatically update the inventory quantity of materials, ensuring the real-time and accuracy of data. The system also provides data statistics functions, intuitively displaying inbound and outbound trends and inventory distribution through charts, helping managers better grasp the warehouse operation status.

After functional testing and actual operation verification, all functions of the system run stably, the interface is easy to operate, and it can meet the basic needs of warehouse management for small and medium-sized enterprises. The design and implementation of this system not only improves the efficiency and accuracy of warehouse management, but also provides a reference solution for the informatization construction of enterprises.

**Key words: Warehouse Management System; SpringBoot; Vue; Front-end and Back-end Separation; Inventory Management**
