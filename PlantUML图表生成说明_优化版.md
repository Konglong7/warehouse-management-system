### 布局优化
- **正方形布局**：所有图表都调整为接近正方形的布局，避免长方形导致的字体不清晰问题

- **线条加粗**：边框和箭头厚度增加到2，提高可读性

- **关系清晰**：只展示关键的调用关系，避免线条交叉混乱

## 如何生成PNG图片

### 方法一：使用在线PlantUML编辑器（最简单）

1. 访问 http://www.plantuml.com/plantuml/uml/
2. 打开对应的 `.puml` 文件，复制全部内容
3. 粘贴到编辑器中，点击"Submit"按钮
4. 等待图片生成（可能需要几秒钟）
5. 右键保存图片，或点击下载按钮
6. 建议保存为PNG格式，分辨率选择高清

### 方法二：使用VS Code插件

1. 在VS Code中安装 "PlantUML" 插件（作者：jebbs）
2. 打开 `.puml` 文件
3. 按 `Alt+D` 预览图表
4. 右键点击预览窗口，选择"Export Current Diagram"
5. 选择PNG格式，设置DPI为300（高清）
6. 保存到指定位置

### 方法三：使用命令行工具

1. 下载 PlantUML jar 文件：
   ```bash
   curl -L -o plantuml.jar https://github.com/plantuml/plantuml/releases/download/v1.2024.3/plantuml-1.2024.3.jar
   ```

2. 生成图片（在项目目录下执行）：
   ```bash
   java -jar plantuml.jar "图3-3_普通用户用例图.puml"
   java -jar plantuml.jar "图3-4_管理员用例图.puml"
   java -jar plantuml.jar "图4-2_系统功能模块图_优化版.puml"
   ```

3. 生成的PNG文件将保存在同一目录下

## 在Word中使用

### 插入图片
1. 生成PNG图片后，在Word中选择"插入" → "图片"
2. 选择对应的PNG文件插入

### 设置图片格式
- **布局**：嵌入型或四周型
- **大小**：根据页面宽度调整
  - A4纸页边距2.5cm，建议图片宽度：14-15cm
  - 保持纵横比，高度自动调整
- **对齐**：居中对齐
- **分辨率**：确保使用高清PNG（300 DPI）

### 添加图注
在图片下方添加图注，格式如下：
- **字体**：宋体，五号（10.5磅），加粗
- **对齐**：居中
- **格式**：图X-X 图表名称

## 图注说明示例

### 图3-3 普通用户用例图

如图3-3所示，普通用户是系统的主要使用者，通常是仓库的操作人员。普通用户拥有11项基本权限，包括登录系统、查看首页统计、修改个人信息、修改登录密码等基础功能，以及查看物料列表、搜索物料信息等查询功能，还可以进行入库操作、出库操作，并查看入库记录、出库记录和库存信息。普通用户不能进行物料、供应商、仓位等基础数据的新增、修改、删除操作，也不能管理用户账户，这样可以避免误操作导致基础数据被破坏。

### 图3-4 管理员用例图

如图3-4所示，管理员是系统的管理者，通常由仓库主管或系统管理员担任。管理员拥有系统的最高权限，共计24项功能权限。除了拥有普通用户的所有权限外，管理员还拥有基础数据管理权限和用户管理权限。在物料管理方面，管理员可以新增、修改和删除物料信息；在供应商管理方面，管理员可以维护供应商档案；在仓位管理方面，管理员可以对仓位进行全面管理；在用户管理方面，管理员可以新增、修改、删除用户账户，并查看所有用户的操作记录。通过这些权限，管理员能够对系统的基础数据进行全面维护，确保系统数据的准确性和完整性。

### 图4-2 系统功能模块图

如图4-2所示，系统采用模块化设计，分为9个功能模块。用户管理模块负责用户认证和权限控制；物料管理模块、供应商管理模块和仓位管理模块构成基础数据管理层；入库管理模块和出库管理模块实现核心业务流程，其中出库操作需要进行库存校验，入库和出库操作都会自动更新库存；库存查询模块和数据统计模块提供数据查询和决策支持功能；系统支撑模块提供数据库管理、日志管理和操作记录等基础服务。各模块之间通过清晰的接口进行交互，形成完整的业务闭环。

## 注意事项

1. **图片清晰度**：建议使用300 DPI的PNG格式，确保打印清晰
2. **图片大小**：正方形布局的图片在Word中显示效果更好
3. **字体大小**：已优化为18-20号字体，截图后无需放大即可清晰阅读
4. **黑白风格**：严格遵循黑白配色，适合打印和复印
5. **线条粗细**：已加粗到2，确保打印后清晰可见

## 常见问题

### Q: 图片在Word中显示模糊怎么办？
A: 确保使用高清PNG格式（300 DPI），不要使用JPG格式。在Word中插入图片后，不要过度放大。

### Q: 如何调整图片大小？
A: 在Word中右键点击图片，选择"大小和位置"，设置宽度为14-15cm，勾选"锁定纵横比"。

### Q: PlantUML在线编辑器生成失败怎么办？
A: 可能是网络问题或代码过长。建议使用VS Code插件或命令行工具生成。

### Q: 图表中文字体显示不正常怎么办？
A: 确保系统已安装"Microsoft YaHei"（微软雅黑）字体。如果没有，可以修改 `.puml` 文件中的 `defaultFontName` 为其他中文字体，如"SimSun"（宋体）。
