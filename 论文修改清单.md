## 一、全局性修改（优先处理）

### 1. Spring Boot写法统一
**查找内容：** `SpringBoot`（不带空格）
**替换为：** `Spring Boot`（中间有空格）
**位置：** 全文所有出现的地方
**说明：** 使用Word的"查找和替换"功能（Ctrl+H），一次性全部替换

**涉及章节：**
- 摘要
- 第二章 相关技术介绍
- 第四章 系统设计
- 第五章 系统实现
- 结论
- 其他所有提到SpringBoot的地方

---

### 2. 标点符号检查
**检查内容：** 确保全文使用中文标点符号
- 中文逗号：，（不是英文逗号,）
- 中文句号：。（不是英文句号.）
- 中文引号：""（不是英文引号""）
- 中文冒号：：（不是英文冒号:）

**位置：** 全文检查

---

### 3. 图表编号检查
**检查内容：** 所有图表是否有规范的编号和标题
- 图的格式：图X-Y（如"图3-1 系统架构图"）
- 表的格式：表X-Y（如"表4-1 用户表结构"）
- 图注在图下方，表注在表上方

**位置：** 全文所有图表

### 【参考文献】

#### 修改15
**位置：** 参考文献部分
**检查内容：**

1. **格式统一性**
   - 是否按照学校要求的格式（通常是GB/T 7714-2015）
   - 作者、标题、出版信息格式是否一致

2. **期刊论文格式示例：**
   ```
   [1] 作者. 文章标题[J]. 期刊名称, 年份, 卷(期): 起始页码-结束页码.
   ```

3. **图书格式示例：**
   ```
   [2] 作者. 书名[M]. 版本. 出版地: 出版社, 年份: 页码.
   ```

4. **网络文献格式示例：**
   ```
   [3] 作者. 文章标题[EB/OL]. (发布日期)[访问日期]. 网址.
   ```

5. **检查要点：**
   - 作者超过3人时，是否用"等"或"et al"
   - 英文文献作者名是否为"姓, 名首字母."格式
   - 网络文献是否标注访问日期
   - 参考文献数量是否符合要求（一般本科论文要求15-30篇）
   - 是否有近3-5年的文献

- 标题字体字号是否符合要求

- 首行缩进是否为2字符

- 目录是否自动生成
