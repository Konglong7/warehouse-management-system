# 基于SpringBoot与Vue的智能仓库管理系统 - 本科毕设极简版

## 一、项目定位

**核心目标**：本地运行 + 答辩演示 + 界面美观 + 功能完整

**不考虑**：高并发、分布式、微服务、复杂权限

## 四、数据库设计（6张核心表）

### 1. 用户表 (sys_user)
```sql
CREATE TABLE sys_user (
    id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '用户ID',
    username VARCHAR(50) NOT NULL UNIQUE COMMENT '用户名',
    password VARCHAR(100) NOT NULL COMMENT '密码',
    nickname VARCHAR(50) COMMENT '昵称',
    phone VARCHAR(20) COMMENT '手机号',
    role VARCHAR(20) DEFAULT 'user' COMMENT '角色：admin/user',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间'
) COMMENT '用户表';
```

### 2. 物料表 (wms_material)
```sql
CREATE TABLE wms_material (
    id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '物料ID',
    material_code VARCHAR(50) NOT NULL UNIQUE COMMENT '物料编码',
    material_name VARCHAR(100) NOT NULL COMMENT '物料名称',
    category VARCHAR(50) COMMENT '物料分类',
    unit VARCHAR(20) COMMENT '单位',
    supplier_id BIGINT COMMENT '供应商ID',
    warning_stock INT DEFAULT 10 COMMENT '预警库存',
    current_stock INT DEFAULT 0 COMMENT '当前库存',
    remark VARCHAR(200) COMMENT '备注',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间'
) COMMENT '物料表';
```

### 3. 供应商表 (wms_supplier)
```sql
CREATE TABLE wms_supplier (
    id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '供应商ID',
    supplier_code VARCHAR(50) NOT NULL UNIQUE COMMENT '供应商编码',

```

### 4. 仓位表 (wms_location)
```sql
CREATE TABLE wms_location (
    id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '仓位ID',
```

## 九、关键代码示例

### 1. 统一返回结果类
```java
@Data
public class Result<T> {
    private Integer code;
    private String message;
    private T data;

    public static <T> Result<T> success(T data) {
        Result<T> result = new Result<>();
        result.setCode(200);
        result.setMessage("success");
        result.setData(data);
        return result;
    }

    public static <T> Result<T> error(String message) {
        Result<T> result = new Result<>();
        result.setCode(500);
        result.setMessage(message);
        return result;
    }
}
```

### 2. 入库业务逻辑
```java
@Service
public class InboundService {
    @Autowired
    private InboundMapper inboundMapper;
    @Autowired
    private MaterialMapper materialMapper;

    @Transactional
    public void addInbound(Inbound inbound) {
        // 1. 保存入库记录
        inbound.setInboundNo("IN" + System.currentTimeMillis());
        inboundMapper.insert(inbound);

        // 2. 更新物料库存
        Material material = materialMapper.selectById(inbound.getMaterialId());
        material.setCurrentStock(material.getCurrentStock() + inbound.getQuantity());
        materialMapper.updateById(material);
    }
}
```

### 3. 前端Axios封装
```javascript
import axios from 'axios'

const request = axios.create({
    baseURL: 'http://localhost:8080',
    timeout: 5000
})

// 请求拦截器
request.interceptors.request.use(config => {
    const token = localStorage.getItem('token')
    if (token) {
        config.headers['Authorization'] = token
    }
    return config
})

// 响应拦截器
request.interceptors.response.use(
    response => response.data,
    error => {
        console.error('请求失败', error)
        return Promise.reject(error)
    }
)

export default request
```
